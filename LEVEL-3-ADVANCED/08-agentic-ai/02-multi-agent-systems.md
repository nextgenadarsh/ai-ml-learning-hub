# Multi-Agent Systems & Coordination

## Overview
This lab introduces multi-agent reinforcement learning (MARL), coordination strategies, environment design, and evaluation metrics for systems with multiple interacting agents.

## Learning Objectives
- Understand MARL paradigms (centralized training, decentralized execution)
- Implement simple cooperative and competitive environments
- Learn common algorithms (MADDPG, QMIX)
- Design reward shaping for team behavior

## Sections
1. MARL concepts and architectures
2. Environment design for multi-agent tasks
3. Example: simple cooperative grid-world with multiple agents
4. Algorithms overview and references

## Exercises
- Implement a cooperative navigation task and train agents with a centralized critic
- Compare independent Q-learning vs joint training
- Evaluate emergent behaviors and metrics
