# Enterprise RAG + Governance Capstone

## Project Brief
Build an end-to-end Retrieval Augmented Generation (RAG) system with vector DB + model governance + monitoring. Track lineage, performance, and compliance, and deliver an executive report.

## Requirements
- Set up vector store (Chroma/Pinecone/Weaviate)
- Implement retrieval + LLM pipeline in FastAPI
- Add logging for data lineage and user queries
- Track cost estimates and horizontal scaling plan
- Add model drift detector + auto-retraining trigger
- Produce governance dashboard (bias, explainability, privacy)

## Deliverables
1. Code repo with architecture diagrams
2. Deployment scripts (Docker/K8s)
3. Monitoring + alerting config (Prometheus/Grafana)
4. Compliance checklist (GDPR/CCPA)
5. Demo notebook and readme

## Assessment
- Architect correctness (30%)
- Reliability & monitoring (25%)
- Governance controls (25%)
- Documentation + presentation (20%)
