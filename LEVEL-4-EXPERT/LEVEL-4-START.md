## 🛠️ Setup & Environment

### Required Tools
```
Python 3.10+
Git & GitHub
Docker & Docker Compose
Kubernetes (basic understanding)
Cloud provider (AWS/GCP/Azure account)
MLflow or Weights & Biases
CI/CD tool (GitHub Actions or GitLab CI)
```

### Infrastructure Setup
```bash
# Docker
curl https://get.docker.com -o get-docker.sh
bash get-docker.sh

# Kubernetes (minikube for local dev)
curl -LO https://github.com/kubernetes/minikube/releases/download/latest/minikube-linux-amd64
chmod +x minikube-linux-amd64

# MLflow
pip install mlflow

# AWS CLI
pip install awscli
aws configure
```
