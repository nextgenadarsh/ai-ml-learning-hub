Switch traffic between versions\n- **Canary**: Gradual rollout to subset\n- **A/B Test**: Compare versions on subsets\n- **Shadow Mode**: Run in parallel without impact\n\n### Caching Patterns\n- **Cache Aside**: Check cache first\n- **Write-Through**: Write to cache and DB\n- **Write-Behind**: Write to cache, async to DB\n\n### Feature Serving Patterns\n- **Offline**: Precomputed features\n- **Online**: Real-time feature lookup\n- **Hybrid**: Combination of both\n\n---\n\n## Common Architectures\n\n### Small Startup (< 10 engineers)\n```\nData Source вҶ’ Pandas/SQL вҶ’ Scikit-learn вҶ’
