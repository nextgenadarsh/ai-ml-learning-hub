# 07-llm-basics

This folder introduces transformer-based models, embeddings, prompt engineering, and practical examples for working with LLMs.

Recommended notebooks and sections:

- `01-transformer-architecture.ipynb` — Attention, encoders/decoders, positional encodings
- `02-llm-models-overview.ipynb` — Common LLMs, model sizes, tradeoffs
- `03-prompt-engineering.ipynb` — Prompt design, chain-of-thought, few-shot
- `04-embedding-models.ipynb` — Sentence embeddings, similarity search
- `05-langchain-basics.ipynb` — Building simple apps with LangChain

How to run:
1. Install optional dependencies from `requirements.txt`
2. Run notebooks locally or in Colab if GPU is needed

Contributions: add practical labs for retrieval-augmented generation and evaluation metrics for LLM outputs.
