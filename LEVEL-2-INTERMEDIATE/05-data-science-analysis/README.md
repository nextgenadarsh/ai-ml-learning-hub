## Key Algorithms

### Clustering
- K-Means: Fast, assumes spherical clusters
- DBSCAN: Density-based, finds arbitrary shapes
- Hierarchical: Creates dendrograms
- Gaussian Mixture Models: Probabilistic approach

### Dimensionality Reduction
- PCA: Linear, variance-based
- t-SNE: Non-linear, visualization-focused
- UMAP: Non-linear, faster than t-SNE
- Autoencoders: Neural network approach

### Text Processing
- Tokenization, Lemmatization, Stemming
- TF-IDF: Statistical approach
- Word2Vec: Dense embeddings
- GloVe: Global word vectors
- BERT: Contextual embeddings
