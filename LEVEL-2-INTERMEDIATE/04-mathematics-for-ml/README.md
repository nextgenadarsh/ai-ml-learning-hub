# Algorithms Module

## Overview
This module covers core algorithms and mathematical foundations for ML and Deep Learning.

## Topics Covered

### 1. Linear Algebra for Machine Learning (01-linear-algebra-for-ml.ipynb)
- Vectors and Matrices
- Matrix Operations
- Eigenvalues and Eigenvectors
- Decomposition Methods (SVD, QR)
- Applications in ML (PCA, Dimensionality Reduction)

**Key Concepts:**
- Dot products and cross products
- Matrix multiplication
- Eigenvalue decomposition
- SVD (Singular Value Decomposition)
- Linear transformations

### 2. Neural Networks and Deep Learning (02-neural-networks.ipynb)
- Perceptrons and Multi-layer Perceptrons
- Activation Functions (ReLU, Sigmoid, Tanh, Softmax)
- Backpropagation Algorithm
- Optimization (SGD, Adam, RMSprop)
- Building DNNs with TensorFlow

**Advanced Topics:**
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN, LSTM, GRU)
- Attention Mechanisms
- Transfer Learning

## Learning Outcomes

After completing this module, you will:
✅ Understand linear algebra concepts fundamental to ML
✅ Grasp how neural networks work
✅ Implement backpropagation
✅ Build deep neural networks
✅ Apply advanced techniques like transfer learning
✅ Debug and optimize neural networks
