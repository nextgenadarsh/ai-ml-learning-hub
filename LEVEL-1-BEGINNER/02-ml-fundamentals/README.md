# Fundamentals Module

## Overview
This module covers the foundational concepts of Machine Learning.

## Topics Covered

### 1. Introduction to Machine Learning (01-intro-to-ml.ipynb)
- What is Machine Learning?
- Types of ML: Supervised, Unsupervised, Reinforcement
- ML Workflow
- Evaluation Metrics

**Key Learnings:**
- Understand ML basics
- Know when to use different ML approaches
- Evaluate models effectively

### 2. Supervised Learning (02-supervised-learning.ipynb)
- Classification Algorithms
- Regression Algorithms
- Performance Metrics

**Algorithms Covered:**
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Linear Regression
- Ridge & Lasso Regression

## Learning Outcomes

After completing this module, you will:
✅ Understand fundamental ML concepts
✅ Know the difference between supervised and unsupervised learning
✅ Build and evaluate classification models
✅ Build and evaluate regression models
✅ Handle training/testing splits and validation
✅ Deal with overfitting and underfitting
✅ Apply regularization techniques
