# Requirements

This file lists all Python packages and their versions needed for the AI/ML Learning Hub course.

## Installation

### Using pip
```bash
pip install -r requirements.txt
```

### Using conda
```bash
conda create -n aiml python=3.10
conda activate aiml
pip install -r requirements.txt
```

## System Requirements

- **RAM**: Minimum 8GB (16GB recommended)
- **Storage**: 10GB for datasets and models
- **GPU**: Optional (CUDA 11.8+ for faster training)
- **OS**: Windows, macOS, or Linux

## Installation Troubleshooting

### TensorFlow Issues
```bash
# GPU support (CUDA 11.8)
pip install tensorflow[and-cuda]==2.13.0

# CPU only
pip install tensorflow-cpu==2.13.0
```

### PyTorch Issues
```bash
# GPU support (CUDA 11.8)
conda install pytorch torchvision pytorch-cuda=11.8 -c pytorch -c nvidia

# CPU only
conda install pytorch torchvision cpuonly -c pytorch
```

### Spacy Models
```bash
python -m spacy download en_core_web_sm
```

### NLTK Data
```python
import nltk
nltk.download('punkt')
nltk.download('averaged_perceptron_tagger')
nltk.download('stopwords')
nltk.download('wordnet')
```

## Updating Packages

```bash
# Update all packages
pip install --upgrade -r requirements.txt

# Update specific package
pip install --upgrade package_name
```

## Verifying Installation

Run the following script to verify all installations:

```python
import sys
print("Python version:", sys.version)

packages = [
    'numpy', 'pandas', 'scipy',
    'sklearn', 'xgboost', 'lightgbm',
    'tensorflow', 'torch',
    'nltk', 'gensim', 'spacy',
    'matplotlib', 'seaborn', 'plotly',
    'jupyter'
]

for package in packages:
    try:
        __import__(package)
        print(f"✓ {package}")
    except ImportError:
        print(f"✗ {package} (not installed)")
```

## Virtual Environment Setup

### Using venv (Python 3.10+)
```bash
python -m venv aiml_env
source aiml_env/bin/activate  # On Windows: aiml_env\Scripts\activate
pip install -r requirements.txt
```

### Using conda
```bash
conda create -n aiml python=3.10
conda activate aiml
pip install -r requirements.txt
```

## Jupyter Kernel Setup

```bash
python -m ipykernel install --user --name aiml --display-name "AI/ML Hub"
```

Then select "AI/ML Hub" kernel when opening notebooks.
